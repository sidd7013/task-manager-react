# 📝 Task Manager (Full Stack)

A full-stack Task Manager application built using **React + Spring Boot + MySQL**.

---

## 🚀 Features

- ➕ Add Task  
- ❌ Delete Task  
- ✔️ Mark Complete / Pending  
- ✏️ Edit Task  
- 🔍 Filter (All / Completed / Pending)  
- 🔄 Real-time UI updates  
- 🌐 Backend API integration  

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Spring Boot
- REST API

### Database
- MySQL

---

## 📂 Project Structure
task-manager-frontend/

## ⚙️ How to Run

### 1️⃣ Backend (Spring Boot)

Run TaskManagerBackendApplication.java


### 2️⃣ Frontend (React)

npm install
npm run dev
